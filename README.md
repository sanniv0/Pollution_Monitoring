# Environmental Pollution Project

## Overview
This project aims to analyze environmental pollution data across various cities using machine learning techniques. The goal is to identify patterns and insights that can help in mitigating pollution.

## Features
- Data analysis and visualization
- Machine learning model for pollution prediction
- Interactive Jupyter Notebook for exploration

## Project Structure
```
├── .gitignore
├── .ipynb_checkpoints/
│   └── environmental_pollution_project_model-checkpoint.ipynb
├── Dataset_Cities.csv
├── environmental_pollution_project_model.ipynb
```

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/sanniv0/Pollution_Monitoring.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Pollution_Monitoring
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
1. Open the Jupyter Notebook:
   ```bash
   jupyter notebook environmental_pollution_project_model.ipynb
   ```
2. Follow the instructions in the notebook to analyze data and train the model.
